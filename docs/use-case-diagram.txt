3. Chức năng của từng Actor:
3.1. Khách vãng lai (Guest)
+ Đăng ký tài khoản
+ Đăng nhập
+ Tìm kiếm và Xem cơ sở sân
+ Nhận đề xuất với AI

+ Quản lý lịch sử đặt sân
+ Quản lý trang cá nhân
+ Đánh giá cơ sở
+ Đăng ký làm chủ sân
3.3. Chủ sân (Court Owner)
+ Xem báo cáo thống kê
+ Quản lý cơ sở
+ Quản lý đơn đặt sân
+ Quản lý khách hàng
+ Quản lý trang cá nhân
3.4. Quản trị viên (Admin)
+ Xem báo cáo thống kê
+ Quản lý người dùng
+ Duyệt đăng ký chủ sân
+ Quản lý doanh thu
